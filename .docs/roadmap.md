# AgentQuilt Roadmap

## Milestone 0.1 — Structured Compilation MVP

Goal:

AgentQuilt can compile structured agent source files into deterministic Markdown and validate them in CI.

Scope:

- Agent manifest schema
- Instruction block schema
- Example agent
- Validate command
- Compile command
- Compile check
- Basic CI workflow
- Golden-file tests

## Milestone 0.2 — Reviewable Agent Changes

Scope:

- Semantic diff
- Risk classification
- PR summaries
- CODEOWNERS
- Gate policy schema
- Eval schema

## Milestone 0.3 — Agent Behavior Testing

Scope:

- Static eval runner
- Eval coverage report
- Regression evals
- Negative evals
- Security evals

## Milestone 0.4 — Governed Agent Lifecycle

Scope:

- Risk register
- Governance checker
- Security scanner
- Release report
- Internal SDLC agents
