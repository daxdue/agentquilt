# ADR-0003: Generated Markdown Is a Build Artifact

## Status

Accepted

## Context

The original problem is caused by manually editing large Markdown agent files. To reduce merge conflicts and improve reviewability, Markdown should not be treated as the primary editable source.

## Decision

Agent Markdown files generated by AgentQuilt are build artifacts. The source of truth is the structured manifest and instruction blocks.

Generated Markdown may be committed to Git for review visibility, but it must not be manually edited.

## Consequences

Positive:

- Reduces direct edits to large Markdown files
- Enables deterministic output checks
- Makes CI validation possible
- Improves review quality

Negative:

- Generated files can still create diffs
- Contributors must learn the source/generation model
- CI must check that generated files are up to date
