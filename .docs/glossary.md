# Glossary

## Agent

A configured AI assistant defined by structured instructions and compiled into a deployable prompt artifact.

## Agent Manifest

The main metadata and configuration file for an agent.

## Instruction Block

A small structured source file representing one behavior, context section, tool rule, output rule, safety rule, example, or constraint.

## Generated Prompt

The compiled Markdown output generated from the manifest and instruction blocks.

## Eval

A test case that validates expected agent behavior.

## Gate

A required quality checkpoint in the SDLC/STLC process.

## Semantic Diff

A review-friendly summary of behavior-impacting changes between two versions of an agent.

## Golden-File Test

A test that compares generated output against a committed expected output file.

## Governance Check

A policy-based validation that confirms ownership, risk, security, and review requirements are satisfied.
